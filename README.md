# MichiHax's Official Texture Pack!

The official MichiHax texture pack. Made with PackGen: https://packgenweb.pages.dev/ by Asterveila.
Note: I have received permission from Asterveila to publish this texture pack.

This texture pack features:

(icons in the screenshot are not included, they are from the Crafting texture pack)

- Green and white colored menus (that include menus in mods!)
- Green and white colored lists
- Green colored demon faces
- A custom game logo that says "MICHIHAX DASH"
- Custom Sliders
- Account profile picture is my profile picture instead of the default one
- goldFont color has been changed to white
- Grey colored gradient background
- MenuLoop has been changed to ":3" by TangerMusic

Mods required:

Texture Loader (https://geode-sdk.org/mods/geode.texture-loader)

And all of this includes a medium port, for the people that need it!

# Installation guide
- Download the "Texture Loader" mod from Geode
- Go to your Geometry Dash 2.207 folder
- Go to the folder called geode > config > geode.texture-loader > packs
- Put the downloaded ZIP there
